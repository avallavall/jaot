# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment of any kind.
Disagreement about code is fine; disrespect toward people is not.

Violations can be reported privately to the maintainer via GitHub
(profile contact on [@avallavall](https://github.com/avallavall)) or through
the repository's private vulnerability reporting form if the matter is
sensitive. Reports are handled confidentially.
