# Introduction

Welcome to JAOT, the self-hosted, multi-tenant optimization platform. JAOT lets you visually build optimization models, run them with industrial-grade solvers, and understand results -- all without writing code.

## What is JAOT?

JAOT provides a complete platform for mathematical optimization:

- **Visual Model Builder** -- Design optimization models with a drag-and-drop interface
- **Industrial Solvers** -- Run models using PySCIPOpt and other state-of-the-art solvers
- **Real-time Results** -- View solutions, sensitivity analysis, and optimization insights
- **Team Collaboration** -- Share models and results across your organization

## Quick Overview

To get started with JAOT, you need to:

1. Create an organization account
2. Set up your API key for programmatic access
3. Build your first optimization model
4. Run a solve and review results
